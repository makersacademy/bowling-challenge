
Bowling Challenge
=================

```
As a player,
So I can play ten pin bowling,
I want to start a game

As a player,
So I can start my game,
I want to roll a ball

As a bad player,
When I play a gutter game,
I want to return my total score

As a bad player,
When I play a gutter game,
I want to see what I bowled in each frame

As a bad player,
When I play a gutter game,
I want to see the total points scored in each frame
```
