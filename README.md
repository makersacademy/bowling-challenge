# bowling-challenge

## User Stories

```
As a player,
If I knock down 10 pins on the first roll,
I expect to receive 10 points and points from my next two rolls as a bonus.
```
```
As a player,
If I roll 12 strikes in a game,
I expect my score to be 300 points for a Perfect Game.
```
