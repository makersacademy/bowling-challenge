Bowling Calculator

To start a game, please press start

Please enter points as an array. See example of format below.

Example: [[1,4],[4,5],[6,4],[5,5],[10],[0,1],[7,3],[6,4],[10],[2,8]],[6]

All regular points in array of arrays. Separate array for one array. If you hit a strike enter [10,0] for that frame. In case of strike or spare in last frame: enter a separate array after the 'regular' array with either one entry( in case of spare) or 2 entries(strike).
